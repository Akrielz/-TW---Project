    • Password [Field]

      • Email [Field]
        • This option may require the use of an Automated Mail API for validation
    • Option 2:
      • Security Question [Field]

    • Register [Button]
      • Error Message If:
        • Passwords not matching
        • Password not having at least 8 characters
        • Password not having at least 1 letter
        • Password not having at least 1 digit
        • Any field exceeds the character limits
        • Invalid Email (Option 1)
      • Else:
        • Hash(Password)
        • Option 1:
          • Set the status "inactive" until mail confirmation
          • Send activation mail to the Email Adress
          • #Any account without email confirmed will be deleted after 24 hours#
        • Option 2:
          • Hash(Security Answer)
        • Create the Account and add it in the Data Base
        • Redirects user to Home Page
